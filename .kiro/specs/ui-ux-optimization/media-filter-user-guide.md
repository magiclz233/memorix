# 媒体库筛选功能使用指南

## 概述

媒体库筛选功能提供了强大而灵活的媒体文件管理能力，支持多维度筛选、URL 分享和高性能虚拟滚动。

---

## 功能特性

### 🔍 搜索功能

- **位置**: 页面顶部搜索框
- **功能**: 按文件名搜索媒体文件
- **特性**: 
  - 实时搜索，防抖 300ms
  - 支持清除按钮
  - 搜索结果高亮显示

**使用方法**:
1. 在搜索框中输入关键词
2. 等待 300ms 后自动搜索
3. 点击 X 按钮清除搜索

---

### 📁 存储类型快速筛选

- **位置**: 搜索框下方
- **选项**: All / Local / NAS / S3
- **功能**: 快速切换不同存储类型的媒体

**使用方法**:
1. 点击存储类型按钮 (All/Local/NAS/S3)
2. 页面自动刷新显示对应存储的媒体
3. 激活的存储类型会高亮显示

**提示**: 
- 选择具体存储类型后，可以进一步选择存储实例
- 切换存储类型会自动清除存储实例筛选

---

### ⚙️ 高级筛选

- **位置**: 搜索框右侧的"高级筛选"按钮
- **功能**: 提供更多筛选维度

**筛选维度**:

#### 1. 存储实例
- 根据选择的存储类型显示对应的实例列表
- 例如: Local #1, Local #2, NAS #1 等

#### 2. 发布状态
- **全部**: 显示所有媒体
- **已发布**: 仅显示已发布的媒体
- **未发布**: 仅显示未发布的媒体

#### 3. 媒体类型
- **全部**: 显示所有类型
- **图片**: 仅显示图片
- **视频**: 仅显示视频
- **动图**: 仅显示 GIF 动图

#### 4. 日期范围
- **开始日期**: 筛选此日期之后的媒体
- **结束日期**: 筛选此日期之前的媒体
- 可以只设置一个日期

#### 5. Hero 标记
- **全部**: 显示所有媒体
- **仅 Hero**: 仅显示标记为 Hero 的媒体
- **非 Hero**: 仅显示未标记为 Hero 的媒体

**使用方法**:
1. 点击"高级筛选"按钮
2. 在弹出的对话框中选择筛选条件
3. 点击"应用"按钮
4. 点击"清除筛选"可重置所有高级筛选

---

### 🏷️ 筛选状态 Chips

- **位置**: 筛选栏下方
- **功能**: 显示当前激活的筛选条件

**特性**:
- 每个筛选条件显示为一个 Chip
- 点击 Chip 上的 X 按钮可单独移除该筛选
- 点击"清除"按钮可移除所有筛选

**示例**:
```
当前筛选: [Local ×] [已发布 ×] [图片 ×] [清除]
```

---

### 🔗 URL 分享功能

所有筛选条件会自动同步到 URL 参数，支持以下功能:

#### 1. 分享筛选结果
- 复制浏览器地址栏的 URL
- 发送给其他用户
- 其他用户打开 URL 后会看到相同的筛选结果

#### 2. 保存筛选状态
- 将 URL 添加到浏览器书签
- 下次打开书签会恢复筛选状态

#### 3. 浏览器导航
- 支持浏览器的前进/后退按钮
- 每次筛选变化都会记录到历史记录

**URL 参数说明**:
```
/dashboard/media?
  q=photo                    # 搜索关键词
  &category=local            # 存储类型
  &storageId=1               # 存储实例 ID
  &status=published          # 发布状态
  &type=image                # 媒体类型
  &dateFrom=2024-01-01       # 开始日期
  &dateTo=2024-12-31         # 结束日期
  &hero=yes                  # Hero 标记
```

## 使用场景

### 场景 1: 查找特定存储的已发布图片

1. 点击存储类型 "Local"
2. 点击"高级筛选"
3. 选择发布状态 "已发布"
4. 选择媒体类型 "图片"
5. 点击"应用"

**结果 URL**:
```
/dashboard/media?category=local&status=published&type=image
```

---

### 场景 2: 查找 2024 年的 Hero 视频

1. 点击"高级筛选"
2. 选择媒体类型 "视频"
3. 设置开始日期 "2024-01-01"
4. 设置结束日期 "2024-12-31"
5. 选择 Hero 标记 "仅 Hero"
6. 点击"应用"

**结果 URL**:
```
/dashboard/media?type=video&dateFrom=2024-01-01&dateTo=2024-12-31&hero=yes
```

---

### 场景 3: 搜索包含 "sunset" 的未发布照片

1. 在搜索框输入 "sunset"
2. 点击"高级筛选"
3. 选择发布状态 "未发布"
4. 选择媒体类型 "图片"
5. 点击"应用"

**结果 URL**:
```
/dashboard/media?q=sunset&status=unpublished&type=image
```

## 性能优化

### 虚拟滚动

- 媒体库使用虚拟滚动技术
- 只渲染可见区域的媒体项
- 支持大量媒体文件 (1000+) 无卡顿

### 图片懒加载

- 图片采用懒加载策略
- 优先加载可见区域的图片
- 使用 BlurHash 占位符提升体验

### 搜索防抖

- 搜索输入防抖 300ms
- 减少不必要的 API 请求
- 提升搜索响应速度

## 快捷操作

### 键盘快捷键

- **Tab**: 在筛选控件之间切换
- **Enter**: 应用筛选 (在高级筛选面板中)
- **Escape**: 关闭高级筛选面板

### 批量操作

1. 选择媒体文件 (点击复选框)
2. 使用批量操作栏进行操作:
   - 发布/取消发布
   - 设置/取消 Hero
   - 批量下载
   - 批量删除

## 常见问题

### Q: 如何清除所有筛选条件?

**A**: 有两种方法:
1. 点击筛选状态 Chips 区域的"清除"按钮
2. 点击"高级筛选"，然后点击"清除筛选"

---

### Q: 为什么切换存储类型后存储实例筛选消失了?

**A**: 这是设计行为。切换存储类型时会自动清除存储实例筛选，因为不同存储类型的实例列表不同。

---

### Q: 如何保存常用的筛选组合?

**A**: 目前可以通过以下方式:
1. 将筛选结果的 URL 添加到浏览器书签
2. 复制 URL 保存到笔记中

未来版本可能会添加筛选预设保存功能。

---

### Q: 筛选响应速度慢怎么办?

**A**: 
1. 检查网络连接
2. 尝试减少筛选维度
3. 使用搜索功能缩小范围
4. 联系管理员检查服务器性能

---

### Q: 移动端如何使用高级筛选?

**A**: 
1. 点击"高级筛选"按钮
2. 在全屏对话框中选择筛选条件
3. 点击"应用"按钮
4. 对话框会自动关闭

## 技术细节

- `AdvancedFilterPanel`: 高级筛选面板
- `FilterChips`: 筛选状态 Chips
- `useMediaFilters`: 筛选状态管理 Hook

- 图片懒加载: 启用

- Firefox 88+
- Safari 14+
